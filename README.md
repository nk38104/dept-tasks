# DEPT tasks

## Task 1

Partition array into 3 groups with approximately same sums.

Example 1:
    arr: [1, 3, 1, 7, 5, 6, 6, 7, 4, 2]
    res: [7, 5, 1, 1], [7, 4, 3], [6, 6, 2] -> sums: 14, 14, 14


Example 2:
    arr: [8, 1, 5, 2, 4, 1, 9, 8]
    res: [9, 2, 1, 1], [8, 4], [8, 5] -> sums: 13, 12, 13
