# DEPT tasks

## Task 1

Partition given array into 3 groups with approximately same sums.
<pre>
Example 1:<br>
    arr: [1, 3, 1, 7, 5, 6, 6, 7, 4, 2]<br>
    res: [7, 5, 1, 1], [7, 4, 3], [6, 6, 2]  ->  sums: 14, 14, 14<br>


Example 2:<br>
    arr: [8, 1, 5, 2, 4, 1, 9, 8]<br>
    res: [9, 2, 1, 1], [8, 4], [8, 5]  ->  sums: 13, 12, 13
</pre>
<br><br>
